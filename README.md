## üì• Cara Instalasi

Pastikan aplikasi pengelola referensi Anda (Zotero atau Mendeley) sudah terbuka sebelum mengikuti langkah-langkah di bawah ini.

---
### 1. Instalasi di Zotero

Ada dua cara untuk memasang gaya ini:
#### Cara A: Melalui Tautan Langsung (Direkomendasikan)
1. Salin tautan "Raw" dari berkas CSL Anda di GitHub (atau klik [DI SINI](PASTE_LINK_RAW_ANDA_DI_SINI)).
2. Jika Anda menggunakan **Zotero Connector** pada browser, Zotero akan secara otomatis mendeteksi file tersebut dan bertanya: *"Install style from URL?"*.
3. Klik **Yes/Install**.
#### Cara B: Instalasi Manual
1. Unduh (download) berkas `.csl` dari repositori ini ke komputer Anda.
2. Buka Zotero, lalu pilih menu **Edit** > **Settings** (atau **Preferences** pada Mac).
3. Pilih tab **Cite** > **Styles**.
4. Klik ikon tambah (**+**) yang terletak di bawah daftar gaya.
5. Cari dan pilih berkas `.csl` yang sudah diunduh tadi, lalu klik **Open**.
---

### 2. Instalasi di Mendeley

#### Mendeley Desktop (Versi Lama)
1. Buka Mendeley Desktop.
2. Pilih menu **View** > **Citation Style** > **More Styles...**.
3. Pilih tab **Get More Styles**.
4. Tempel (Paste) tautan "Raw" berkas CSL Anda (contoh: `https://raw.githubusercontent.com/.../style.csl`) ke dalam kotak **Download URL**.
5. Klik **Download**.
6. Setelah selesai, buka tab **Installed**, cari gaya tersebut, lalu klik **Use this Style**
#### Mendeley Reference Manager (Versi Baru)
1. Buka Mendeley Reference Manager.
2. Masuk ke tab **Citation Settings**.
3. Klik tombol **Change citation style**.
4. Klik **Add a custom style**.
5. Tempel (Paste) tautan "Raw" berkas CSL Anda ke dalam kotak yang tersedia.
6. Klik **Update citation style**.
---

## üìù Catatan Penting
* **Tautan Raw:** Pastikan Anda selalu menggunakan tautan yang diawali dengan `https://raw.githubusercontent.com/...`. Tautan halaman GitHub biasa tidak akan berfungsi untuk instalasi langsung.
* **Pembaruan:** Jika terjadi perubahan atau perbaikan pada berkas CSL ini, Anda perlu mengulang langkah instalasi untuk mendapatkan versi terbaru.
